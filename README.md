# Homebrew Tap for HexClaw

## Install

```bash
brew tap hexagon-codes/tap
brew install --cask hexclaw
```

## Update

```bash
brew upgrade --cask hexclaw
```

## Uninstall

```bash
brew uninstall --cask hexclaw
```

## About

[HexClaw Desktop](https://github.com/hexagon-codes/hexclaw-desktop) is an enterprise-grade personal AI Agent desktop client built with Tauri v2 + Vue 3.
